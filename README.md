# Sistema de Alunos - Visualg

Projeto desenvolvido para praticar lógica de programação utilizando o Visualg.

## 📚 Sobre o projeto

Este algoritmo simula um pequeno sistema de gerenciamento de alunos no terminal.  
O programa permite cadastrar alunos, armazenar notas e realizar consultas utilizando vetores.

## ⚙️ Funcionalidades

- Cadastro de alunos
- Armazenamento de nomes e notas em vetores
- Listagem de alunos cadastrados
- Cálculo da média da turma
- Exibição de alunos aprovados
- Busca de aluno pelo nome

## 🧠 Conceitos praticados

- Vetores
- Estruturas de repetição
- Estruturas condicionais
- Menu interativo
- Lógica de programação

## 💻 Tecnologias utilizadas

- Visualg

## ▶️ Como executar

1. Abra o arquivo `.alg` no Visualg.
2. Execute o algoritmo.
3. Utilize o menu para testar as funcionalidades do sistema.
